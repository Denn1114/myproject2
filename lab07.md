\# Документація тестування



\## Запуск тестів



\### Всі тести

pytest



\### З покриттям коду

pytest --cov=app --cov-report=html



\## Опис тестових сценаріїв



\### Unit тести



\#### test\_add\_positive\_numbers

Перевіряє додавання двох додатних чисел.

Очікуваний результат: сума чисел.



\#### test\_divide\_by\_zero

Перевіряє обробку ділення на нуль.

Очікуваний результат: виняток ZeroDivisionError.



\#### test\_feedback\_creation

Перевіряє створення нового відгуку через функцію add\_feedback().

Очікуваний результат: відгук успішно доданий у базу даних.



\#### test\_order\_total\_calculation

Перевіряє правильність обчислення загальної суми замовлення.

Очікуваний результат: сума всіх товарів у замовленні відповідає очікуваній.



\### Integration тести



\#### test\_create\_user

Перевіряє створення нового користувача через API.

Очікуваний результат: статус 201, об'єкт користувача з ID.



\#### test\_get\_all\_feedbacks

Перевіряє отримання списку всіх відгуків через API.

Очікуваний результат: повертається список відгуків з правильними даними.



\#### test\_order\_workflow

Перевіряє повний процес замовлення: додавання товару до кошика, оформлення замовлення, зміна статусу.

Очікуваний результат: всі кроки виконані успішно, статус замовлення змінюється коректно.
