\# Fan Shop Barça — README



\## Опис проєкту та його призначення

Fan Shop Barça — це вебзастосунок для фанатів футбольного клубу Барселона.  

Він дозволяє користувачам переглядати каталог товарів, купувати футболки, шарфи та аксесуари, залишати відгуки та керувати своїми замовленнями.  

Для адміністратора доступна панель керування користувачами та відгуками.



\## Використані технології та обґрунтування вибору

\- \*\*Python 3.x\*\* — мова програмування для бекенду, проста у використанні та підтримує Flask.

\- \*\*Flask\*\* — легкий фреймворк для створення вебзастосунків.

\- \*\*SQLite\*\* — проста реляційна база даних для зберігання користувачів, відгуків, замовлень та товарів.

\- \*\*HTML/CSS/JavaScript\*\* — для фронтенду та інтерактивності інтерфейсу.

\- \*\*Flask-Login\*\* — керування автентифікацією користувачів.

\- \*\*Werkzeug\*\* — хешування паролів та безпека.

\- \*\*Swagger (Flasgger)\*\* — документація API.



\## Архітектура системи

Система поділена на кілька компонентів:

\- \*\*Frontend\*\* — шаблони HTML, CSS та JS.

\- \*\*Backend (Flask)\*\* — маршрути, логіка бізнес-процесів.

\- \*\*База даних (SQLite)\*\* — таблиці користувачів, відгуків, замовлень та товарів.

\- \*\*Адмін-панель\*\* — доступна лише адміністраторам для керування користувачами та відгуками.

\- \*\*API\*\* — дозволяє працювати з даними через REST-запити.
