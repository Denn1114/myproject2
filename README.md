# Лабораторна робота №5: Розробка RESTful API

## Інформація про проєкт
- **Назва проєкту:** MyShop API
- **Автори:** Мазурок денис

## Опис проєкту
API для управління клієнтами та замовленнями у навчальному магазині. Дозволяє отримувати, створювати та переглядати записи клієнтів та замовлень.

## Технології
- Python 3.x
- Flask
- SQLite
- Marshmallow
- Flasgger (Swagger UI)
- Postman

## Endpoints API

### 1. Отримати список клієнтів
- **URL:** `/api/v1/clients`
- **Метод:** GET
- **Опис:** Повертає список усіх клієнтів
- **Приклад відповіді:**
```json
[
  {"id":1, "name":"Ivan", "email":"ivan@mail.com"},
  {"id":2, "name":"Olena", "email":"olena@mail.com"}
]
